Introduction

Building Salesforce features in the AI era requires more than just chat interaction. It requires Rigor.

emergency The SDD Strategy

SFSpeckit is a Spec-Driven Development framework that enforces Requirement > Design > Build discipline. It eliminates AI hallucinations by locking and gating every phase of the lifecycle.

Installation

Step 1: Add the extension to your Spec Kit installation:

specify extension add sf --from https://github.com/ysumanth06/spec-kit-sf/archive/refs/tags/v1.0.0.zip

Step 2: Initialize the automated environment setup:

/speckit.sf.setup
Auto-Checker: Detects and configures Salesforce CLI, GitHub CLI, and Code Analyzer.
